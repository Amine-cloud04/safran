# Safran RH & Formation - Plateforme Complète

Plateforme intégrée pour Safran combinant un chatbot RH intelligent et une solution d'analyse automatisée des évaluations de formation.

## Démarrage Rapide

### Installation
pip install -r requirements.txt

### Lancer le Backend (FastAPI)
python backend_server.py

### Lancer le Frontend (Streamlit)
streamlit run app.py

## Fonctionnalités

### 1. Chatbot RH Intelligent
- Multilingue (Français, Darija)
- Adaptation aux profils (CDI, CDD, Intérim, Stagiaire, Apprenti)
- Réponses instantanées < 2 secondes
- Sécurité: Détection automatique des questions interdites

### 2. Analyse Automatisée des Évaluations
- Analyse quantitative
- Analyse de sentiment
- Extraction de thèmes
- Clustering des commentaires
- Détection de signaux faibles
